# 🍽️ Recipe Website

A responsive and modern Recipe Website built using React.js that allows users to search and explore recipes with detailed instructions.

## 🚀 Features
- 🔍 Search recipes by name  
- 📜 View detailed cooking instructions  
- ⏱️ Display cooking time  
- 🎨 Responsive UI with Tailwind CSS  
- ⚡ Fast performance using Vite  

## 🛠️ Tech Stack
- React.js  
- Tailwind CSS  
- Vite  
- Meal API
